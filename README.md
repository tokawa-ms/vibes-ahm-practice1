# Retro Tetris - 80年代レトロテトリス

このリポジトリは、80年代のレトロゲームスタイルで作られたテトリスゲームです。GitHub Copilot を使用して開発されたデモアプリケーションです。

## 🎮 ゲームをプレイ

**[🎮 テトリスゲームを開始する](src/index.html)**

## ✨ 特徴

- **80年代レトロスタイル**: ネオンカラーとピクセルアートで懐かしの雰囲気を演出
- **コロベイニキ BGM**: ロシア民謡「コロベイニキ」をWeb Audio APIで生成・再生
- **フル機能テトリス**: 回転、ライン消去、スコアシステム、レベルアップ機能
- **レスポンシブデザイン**: デスクトップとモバイルの両方に対応
- **レトロエフェクト**: スキャンライン、CRTモニター効果、グロー効果

## 🎵 BGM について

ゲーム中には有名なテトリスのBGM「コロベイニキ」が流れます。この楽曲はロシアの民謡で、Web Audio APIを使用してリアルタイムで生成されています。

## 🎮 操作方法

- **← →** : テトロミノの移動
- **↓** : 高速落下
- **↑** : 回転
- **SPACE** : 一時停止/再開
- **R** : ゲームリスタート

## 🛠 技術スタック

- **HTML5**: ゲーム画面の構造
- **CSS3 + Tailwind CSS**: レトロスタイルのデザイン
- **JavaScript (ES6)**: ゲームロジックとWeb Audio API
- **Canvas API**: ピクセルパーフェクトなゲーム描画
- **Web Audio API**: リアルタイムBGM生成

## 📁 ファイル構成

```
src/
├── index.html      # メインゲームページ
├── css/
│   └── styles.css  # レトロスタイルCSS
└── js/
    └── script.js   # テトリスゲームロジック
```

## 🚀 実行方法

1. `src/index.html` をブラウザで直接開く
2. またはローカルサーバーを起動して実行

ブラウザがWeb Audio APIをサポートしている必要があります（現代的なブラウザであれば問題ありません）。

## 🎯 ゲーム仕様

- **ボードサイズ**: 10×20
- **テトロミノ**: 全7種類（I, O, T, S, Z, J, L）
- **スコアシステム**: ライン消去数とレベルに応じたスコア計算
- **レベルアップ**: 10ライン消去ごとにレベルアップ、落下速度増加
